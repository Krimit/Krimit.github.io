My Projects

This site

I hacked this website together using Jekyll and GitHubPages in a couple of days. Source viewable at https://github.com/Krimit/Krimit.github.io

NabbTix

I am a co-founder at NabbTix, an exciting startup aimed at making getting tickets to hot events easier. We believe that the current system, where a huge portion of tickets are sold on the secondary market, is flawed. We are researching ways to keep more ticket sales in the primary market.

Mars

A 3D model of mars created in WebGL using Three.js showing the locations of spacecraft landings. Go here to see a live demo. You can check out the extremely messy source code at github.com/Krimit/mars

Blog

A blogging platform coded in Python and hosted on Google AppEngine. I had a great time building it. Code at github.com/Krimit/blog

Connect Four

An implementation of Connect Four in Java. This project was an excercise in using The Observer design pattern (MVC). Created as an assignment for the incredibly awesome Production Quality Software course at NYU with Professor Michael Schidlowsky. Code at https://github.com/Krimit/connect-four
