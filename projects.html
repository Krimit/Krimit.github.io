My Projects

These are some of my recent projects. The source code for many of them can be seen on my github account.

May 14 - Auction Algorithms

An implementation paper in which I implemented and compared 7 different auction algorithms for multiple items. Implementation in Java.

May 14 - Finding Strongly Connected Components

An implementation paper in which I implemented divide and conquer algorithms for finding strongly connected components in parallel. Implementation in Scala.

May 14 - Weather prediction from Tweets

A machine learning paper based on a Kaggle competition, written together with the excellent Bryan Ball. Used a variety of algorithms to predict weather and sentiment from weather-related tweets.

December 14 - This site

I hacked this website together using Jekyll and GitHubPages. Source viewable at https://github.com/Krimit/Krimit.github.io

October 13 - NabbTix

I am a co-founder at NabbTix, an exciting startup aimed at making getting tickets to hot events easier. We believe that the current system, where a huge portion of tickets are sold on the secondary market, is flawed. We are researching ways to keep more ticket sales in the primary market.

April 13 - Mars

A 3D model of mars created in WebGL using Three.js showing the locations of spacecraft landings. Go here to see a live demo. You can check out the extremely messy source code at github.com/Krimit/mars

December 13 - Blog

A blogging platform coded in Python and hosted on Google AppEngine. I had a great time building it. Code at github.com/Krimit/blog

November 13 - Connect Four

An implementation of Connect Four in Java. This project was an exercise in using The Observer design pattern (MVC). Created as an assignment for the incredibly awesome Production Quality Software course at NYU with Professor Michael Schidlowsky. Code at https://github.com/Krimit/connect-four

December 13 - Paint

An implementation of a paint program in Java. This project was an exercise in using The Observer design pattern (MVC). Also an assignment for the fantastic Production Quality Software course at NYU. Code at https://github.com/Krimit/my-paint
